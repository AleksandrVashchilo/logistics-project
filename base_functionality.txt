Базовый функционал:

Запрос: 
	Делается запрос по входным параметрам на возможность открытия заказа на грузоперевозку. По каждому из 6-ти параметров (денежная сумма, расстояние, общий объем, общий вес, класс опасности груза, температурный режим) должен быть получен положительный ответ по всем пунктам для последующего формирования заказа. Эталонные параметры для сравнения будут храниться в файле variables.properties 
    Функция записи запроса записываются в БД.
    Функция просммотра запроса.
    Функция агрегации данных (получение ответа на запрос).


variables.properties:
	В эталонных параметрах хранятся такие данные, как: себестоимость 1 км, полезный объем и вес автотранспортного средства, класс опасности груза и температурный режим, который может быть обеспечен при грузоперевозке.

Вспомогательные функции, Клиенты:
	Функция просмотра имеющихся клиентов с полями (название компании, контактное лицо, адрес, страна, компания действующая или закрыта).
	Функция записи/заведения нового клиента.
	Функция обновления данных.
Вспомогательные функции, Менеджеры:
	Функция просмотра работающих менеджеров с полями (имя сотрудника, позиция/должность, работает или уволен).
	Функция записи/заведения нового сотрудника.
	Функция обновления данных.
Вспомогательная функция, Транспортное средство:
	Функция просмотра транспортных средств с полями (имя водителя, сотовый телефон водителя, рег. номер транспортного средства,  грузоподъёмность/вес макс. груза, объем макс. груза, количество паллета-мест макс. ,транспортное средство действующее или уже нет.
	Функция записи.
	Функция обновления.
Функция оформления груза:
	Функция просмотра параметров груза с полями (наименование/характер, класс опасности, вес, объем, количество паллета-мест, стоимость грузоперевозки, груз принят к перевозке/внесен в заказ или нет).
	Функция записи.
	Функция обновления.
Функция оформления заказа:
	Функция просмотра заказов с полями с элементами агрегации (Id запрос, Id клиента, Id транспортного средства, дата и время создания заказа, заказ исполнен или нет, заказ помечен на удаление или нет).
	Функция записи.
	Функция обновления.
